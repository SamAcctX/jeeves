# Jeeves Documentation

This directory contains comprehensive documentation for the Jeeves container management system and Ralph Loop workflow.

## Purpose

The documentation provides:
- Usage guides for Jeeves and Ralph Loop
- Command references
- Configuration instructions
- Troubleshooting information
- Best practices

## Key Files

| File | Purpose |
|------|---------|
| `commands.md` | Detailed reference for Jeeves and Ralph commands |
| `configuration.md` | Configuration options for Jeeves and Ralph Loop |
| `troubleshooting.md` | Common issues and solutions |
| `agent-selection-guide.md` | Guide to selecting and configuring agents |
| `model-mapping-guide.md` | Model selection and mapping for agents |
| `phase2-decomposition-guide.md` | Task decomposition best practices |
| `example-walkthrough.md` | Step-by-step example using Ralph Loop |

## Getting Started

For new users, start with:
1. `example-walkthrough.md` - Step-by-step project example
2. `commands.md` - Command reference
3. `configuration.md` - Configuration options

For troubleshooting, check:
- `troubleshooting.md` - Common issues and solutions

## Usage

Documentation is available in the container at:
```
/proj/jeeves/docs/
```

You can view it directly with:
```bash
cat /proj/jeeves/docs/commands.md
# Or use a text editor
vi /proj/jeeves/docs/configuration.md
```
